# AI Access Cost Calculator

A small web tool inspired by my Product Bites post on the growing cost of staying professionally relevant in the age of AI.

This calculator helps visualise the monthly and yearly cost of common paid AI tools, making a simple but important question more tangible:

**What does it cost to “stay in the room”?**

## Why I built this

In many industries, AI fluency is increasingly treated as a baseline expectation. But the tools that make people faster, sharper, and more productive often sit behind paid subscriptions.

This project turns that tension into something visible and interactive.

It was inspired by my Product Bites Spark:
**EP07 — Product Ethics**

## Features

- Select common paid AI tools
- See total monthly cost
- See total yearly cost
- Reflect on the cumulative cost of professional relevance

## Tech stack

- HTML
- CSS
- JavaScript

No framework. No backend. Kept intentionally simple.

## Goal

This is a lightweight, thoughtful prototype designed to bring a product ethics question to life in a clear and accessible way.
